# Bella Moda – Projeto SQL

Este repositório contém os scripts SQL desenvolvidos para o projeto Bella Moda, seguindo as orientações da atividade de Criação e Manipulação de Dados.

## Arquivos do projeto

### 1️⃣ 1-inserts.sql
Contém todos os comandos de inserção de dados nas tabelas:
- usuário
- cliente
- categoria
- produto
- venda
- item_venda

### 2️⃣ 2-selects.sql
Contém consultas SQL utilizando:
- SELECT
- WHERE
- ORDER BY
- LIMIT
- JOIN

### 3️⃣ 3-updates_deletes.sql
Contém:
- 3 comandos UPDATE com condição
- 3 comandos DELETE com condição

## Banco de Dados utilizado
USE bella_moda;

## Como executar
1. Abrir o MySQL Workbench.
2. Selecionar a conexão.
3. Executar os arquivos na seguinte ordem:
   - 1-inserts.sql  
   - 2-selects.sql  
   - 3-updates_deletes.sql
